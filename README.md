## Demo

https://xiqi.github.io/splash-cursor-effect-vanilla-js/demo/

## Usage

1. Copy `splashCursor.js` to your project
2. Include the script in the `<head>` section:
```
<script src="splashCursor.js" defer></script>
```
3. Done
